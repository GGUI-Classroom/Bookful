How to Use Bookful

Create checkouts first, then organize students, books, and the account portal

This page walks you through the fastest workflow: start a checkout, manage roster-based student accounts, and stay on top of overdue books.

Step 1

Start with a checkout

Click New Checkout first. Pick the student and book if they already exist, then confirm.

Step 2

Add students by hand or import a roster

Use the student page to add one student at a time or paste or upload a roster for the whole class.

Step 3

Review dashboard alerts

Use the dashboard to spot overdue items and quickly see current classroom lending activity.

Step 4

Mark returns immediately

When books come back, use Mark as Returned so records stay accurate and current.

Step 5

Use the account portal and records together

Students join through the account portal, see their own checkouts, and review the class collection while teachers use Records to import history and track accountability.

Teacher Tip

If the checkout date does not match your local date, update it before confirming. Bookful asks for confirmation so you can catch timezone mismatches before saving.
