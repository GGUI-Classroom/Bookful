How to Use Bookful

Classroom library management in 5 simple steps

This page walks you through setup, daily checkout workflow, and how to stay on top of overdue books.

Step 1

Create your starter list

Go to Students and Books, then add your class and library collection once.

Step 2

Create a checkout

Click the New Checkout button, select student and book, set the checkout date, then confirm.

Step 3

Review dashboard alerts

Use the dashboard to spot overdue items and quickly see current classroom lending activity.

Step 4

Mark returns immediately

When books come back, use Mark as Returned so records stay accurate and current.

Step 5

Use records for accountability

Open Records to view active vs previous checkouts and track long-term borrowing patterns.

Teacher Tip

If the checkout date does not match your local date, update it before confirming. Bookful asks for confirmation so you can catch timezone mismatches before saving.
